# Operon Docs

The complete documentation for Operon, a task and project management system that unifies inline tasks and file-based tasks in the same workflows, with filters, Calendar planning, Kanban boards, recurrence, pinned tasks, and time tracking, inside Obsidian and built for humans and agents alike.

New to Operon? Read [[DOCS-002 How to use these docs|How to use these docs]] first, then follow the start path below.
