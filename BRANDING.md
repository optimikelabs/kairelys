# Kairélys identity

Kairélys uses a separate identity from Operon in accordance with the upstream trademark policy.

- Product name: **Kairélys**
- Obsidian plugin ID: `kairelys`
- Repository: `optimikelabs/kairelys`
- Primary interface mark: the `sparkles` symbol
- Visual direction: midnight violet, cyan and warm starlight
- Attribution line: **Temporary, unofficial fork of Operon. Not endorsed by the Operon maintainer.**

The name combines *kairos* — the right moment for action — with ÉLYSIA. It describes the fork's
purpose without suggesting that it is an official Operon edition.

The words “Operon” and `operonId` remain only where ancestry, data compatibility or the upstream API
contract requires them. Official Operon logos, icons, release banners and artwork are not Kairélys
branding and must not be reused in Kairélys releases.
